Preview ｜ 快速预览指南

本地一键预览，随时调试落地页体验

通过轻量化的本地服务器指令，即可实时预览落地页动画与布局。支持热更新与移动端调试，确保设计体验与交互一致性。

方法一：使用 VS Code + Live Server

在 VS Code 中安装 Live Server 插件，右键 index.html，选择 “Open with Live Server”，即可开启实时预览与热更新。

方法二：命令行快速预览

如果你习惯命令行，可在项目根目录执行以下指令：

python3 -m http.server 5173

打开浏览器访问 http://localhost:5173，即可实时查看页面效果。支持手机扫描同一局域网地址进行移动端预览。
